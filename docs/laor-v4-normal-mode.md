# 라오어 무한매수법 V4.0 교본 - 일반모드

> 내부 프로젝트 정리 문서입니다. 사용자가 제공한 일반모드 자료를 교본 형태로 재구성했습니다. 외부 배포용 문서가 아닙니다.

## 범위

이 문서는 라오어 무한매수법 V4.0의 **일반모드**만 다룹니다.

일반모드는 자금 소진 전까지의 매수, 쿼터매도, 최종매도 과정을 뜻합니다. 소진모드는 진행 회차 `T` 기준으로 1회분 미만이 남은 시점부터 시작하며, 특정 조건을 만족해 다시 일반모드로 돌아가기 전까지의 과정입니다. 소진 후 리버스모드는 [laor-v4-reverse-mode.md](./laor-v4-reverse-mode.md)에서 정리합니다.

## 대상 종목

공식 운용 대상은 다음 두 종목으로 제한합니다.

- `TQQQ`
- `SOXL`

초심자는 `TQQQ`부터 시작하는 것을 기본으로 봅니다. `SOXL`은 일반적으로 수익 기회가 더 클 수 있지만 변동성이 더 커서 중도 포기 가능성도 커집니다.

## 원금과 분할 설정

전략 시작 전에 전용 원금을 정합니다. 매수 후 잔금이 남더라도 이 잔금은 해당 전략의 예비 현금으로 유지하고 다른 투자에 사용하지 않습니다.

추천 분할 수:

- `20분할`: 공격적, 위험도 높음
- `30분할`: 중간 성향
- `40분할`: 방어적, 안정성 높음

`TQQQ`와 `SOXL`을 동시에 운용할 때는 각 종목의 원금을 독립적으로 설정합니다. 종목이 다르면 같은 계좌에서 진행해도 괜찮지만, 같은 종목을 서로 다른 분할 수로 운용할 때는 계좌를 분리하는 편이 낫습니다.

## 핵심 변수

| 변수 | 의미 |
| --- | --- |
| `N` | 전체 분할 수. 예: `20`, `30`, `40` |
| `T` | 진행 회차. `1.0T`는 1회분 진행 상태 |
| `cash` | 전략에 남아 있는 현금 |
| `avg` | 평균 매수가 |
| `qty` | 현재 보유 수량 |
| `targetProfitPercent` | `TQQQ = 15%`, `SOXL = 20%` |

## T값

`T`는 Turn의 약자로 진행 회차를 뜻합니다.

일반모드에서의 T 변화:

| 이벤트 | T 변화 |
| --- | --- |
| 1회 매수 체결 | `T = T + 1` |
| 절반 매수 체결 | `T = T + 0.5` |
| 쿼터매도 체결 | `T = 직전 T * 0.75` |
| 지정가매도 후 LOC 1회 매수 체결 | `T = 직전 T * 0.25 + 1` |
| 지정가매도 후 LOC 절반 매수 체결 | `T = 직전 T * 0.25 + 0.5` |

`T`는 소수점 자리수를 제한하지 않습니다. 계산 결과가 긴 소수여도 그 값 자체를 사용합니다.

## 별%와 별지점

별지점은 매수와 매도의 기준 가격입니다.

일반식:

```text
별% = 목표수익률 * (1 - (2 * T / N))
별지점 = 평균단가 * (1 + 별% / 100)
```

종목과 분할 수별 예:

| 종목 | 분할 수 | 별% 공식 |
| --- | --- | --- |
| `TQQQ` | `20분할` | `15 - 1.5T` |
| `TQQQ` | `40분할` | `15 - 0.75T` |
| `SOXL` | `20분할` | `20 - 2T` |
| `SOXL` | `40분할` | `20 - T` |

`T`가 전체 분할 수의 절반을 지나면 후반전입니다. 후반전에서는 별%가 마이너스가 될 수 있고, 별지점이 평균단가보다 낮아질 수 있습니다.

매수와 매도 가격이 겹치지 않도록 다음 기준을 사용합니다.

- 매수점: `별지점 - 0.01`
- 매도점: `별지점`

## 일일 매수금

첫 매수 시도액:

```text
최초 1회 매수금 = 원금 / N
```

두 번째 매수부터는 남은 현금과 현재 T값으로 매일 다시 계산합니다.

```text
일일 1회 매수금 = cash / (N - T)
```

이 계산은 전일 T 변화가 `+1`, `+0.5`, `*0.75` 등 어떤 방식이었는지와 무관하게 매일 적용합니다.

## 매수 규칙

일반모드의 매수는 모두 `LOC`를 사용합니다.

### 첫 매수

조건:

```text
qty == 0
T == 0
```

규칙:

- 무조건 1회 매수를 시도합니다.
- 전일 종가보다 충분히 큰 가격으로 LOC 매수를 시도합니다.
- 자료 기준으로는 일반적으로 전일 종가보다 `10% ~ 15%` 위 가격을 사용합니다.

### 전반전 매수

조건:

```text
1 <= T < N / 2
```

예를 들어 `40분할`이면 `T = 1`부터 `20` 미만까지가 전반전입니다.

규칙:

- 일일 1회 매수금의 절반은 `별지점 - 0.01`에 LOC 매수합니다.
- 나머지 절반은 `평균단가`에 LOC 매수합니다.
- 큰 하락에 대비해 아래 가격대에도 LOC 매수를 추가해 일일 매수금이 사용될 수 있게 합니다.

### 후반전 매수

조건:

```text
N / 2 <= T <= N - 1
```

예를 들어 `40분할`이면 `T = 20`부터 `39`까지가 후반전입니다.

규칙:

- 일일 1회 매수금 전체를 `별지점 - 0.01`에 LOC 매수합니다.
- 큰 하락에 대비해 아래 가격대에도 LOC 매수를 추가해 일일 매수금이 사용될 수 있게 합니다.

`T > N - 1`이 되면 일반모드를 벗어나 소진모드로 넘어갑니다.

## 매도 규칙

매도는 전반전과 후반전에 동일하게 적용합니다.

매일 다음 두 종류의 매도를 함께 겁니다.

1. 쿼터매도
2. 최종 지정가매도

### 쿼터매도

규칙:

- 현재 보유 수량의 `1/4`을 매도합니다.
- 주문 타입은 `LOC`입니다.
- 가격은 `별지점`입니다.

### 최종 지정가매도

규칙:

- 쿼터매도 대상이 아닌 나머지 보유 수량을 매도합니다.
- 주문 타입은 지정가매도입니다.
- 가격은 평균단가 기준 목표수익률입니다.

```text
TQQQ 최종매도가 = avg * 1.15
SOXL 최종매도가 = avg * 1.20
```

지정가매도는 프리장, 본장, 애프터장 기회를 모두 활용할 수 있도록 가능한 한 이른 시간에 거는 것을 원칙으로 봅니다. LOC 매수와 LOC 매도는 예약으로 미리 걸어도 되고, 본장에서 걸어도 됩니다.

주간장은 사용하지 않습니다.

## 종료 시점

한 회차는 다음 조건에서 종료됩니다.

```text
qty == 0
```

지정가매도 후 가격이 크게 하락해 LOC 매수가 함께 체결된 경우에는 보유 수량이 다시 생기므로 종료가 아닙니다. 이 경우 기존 회차를 계속 이어갑니다.

종료 후에는 첫 매수 단계로 돌아갑니다. 다음 회차 원금에 수익금을 포함할지는 운용 정책입니다.

- 복리: 수익금을 다음 회차 원금에 포함
- 단리: 최초 원금을 유지하고 수익금은 분리

## 큰수매수

큰수매수는 무한매수법의 핵심 규칙이라기보다 증권사 주문 거부를 피하기 위한 실행 보조 개념입니다.

일부 증권사는 현재가와 주문 가격 차이가 크면 매매 실수 방지 차원에서 주문을 거부할 수 있습니다. 3배 레버리지 종목은 급락장에서 이 문제가 자주 발생할 수 있으므로, 주문 거부를 피하기 위해 큰수매수 개념을 사용합니다.

큰수매수의 상세 규칙은 별도 문서로 분리합니다.

## 이후 정리 대상

다음 항목은 별도 교본 또는 운영 절차로 분리합니다.

- 큰 하락 대비 하단 LOC 매수 사다리 생성 규칙
- 큰수매수 상세 규칙
- 지정가매도 후 LOC 매수 동시 체결 시 T값 계산
- 증권사별 프리장, 본장, 애프터장 주문 유효 범위
- 소진 후 리버스모드의 세부 체결 순서와 백테스트 적용 방식
